BaseRcvAdapter

abstract class BaseRcvAdapter<ITEM : Any, VH : RecyclerView.ViewHolder>(testExecutor: Executor? = null) : RecyclerView.Adapter<VH>

RecyclerView base adapter backed by AsyncListDiffer for background DiffUtil calculation.

백그라운드 DiffUtil 계산을 위해 AsyncListDiffer를 사용하는 RecyclerView 기본 어댑터입니다.

Type of items in the adapter.

어댑터에서 사용할 뷰홀더 타입입니다.

testExecutor
