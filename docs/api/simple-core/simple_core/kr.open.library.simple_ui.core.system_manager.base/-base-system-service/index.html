abstract class BaseSystemService(context: Context, requiredPermissions: List<String>? = null)

Base class for system services with integrated Result pattern support. Result 패턴 지원이 통합된 시스템 서비스의 기본 클래스입니다.

This class provides: 이 클래스는 다음을 제공합니다:

  • Standardized error categorization / 표준화된 오류 분류
