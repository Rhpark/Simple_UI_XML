createWaveform

@RequiresPermission(value = "android.permission.VIBRATE")
fun createWaveform(times: LongArray, amplitudes: IntArray, repeat: Int = -1): Boolean

Creates a complex waveform vibration pattern with custom timing and amplitudes.
Supports different APIs based on SDK version for backward compatibility.

사용자 정의 타이밍과 강도로 복잡한 웨이브폼 진동 패턴을 생성합니다.
하위 호환성을 위해 SDK 버전에 따라 다른 API를 지원합니다.

Return

true if vibration was triggered successfully, false otherwise.

진동 실행 성공 시 true, 그렇지 않으면 false.

Parameters

times

Timing values in milliseconds.

타이밍 값 (밀리초 단위).

amplitudes

Amplitude values (0-255) where 0 = motor off.

강도 값 (0-255), 0은 모터 끔.

repeat

Index to start repeating from, -1 for no repeat.

반복 시작 인덱스, -1은 반복 없음.
