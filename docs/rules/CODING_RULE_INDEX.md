# 코딩 규칙 인덱스

코드 작성 시 항상 준수해야 할 규칙 문서 목록입니다.
상세 내용은 `docs/rules/coding_rule/` 하위 파일을 참조합니다.

## 사용 원칙

- 단계 문서(code_review, code_refactor)에서는 이 인덱스를 규칙 기준 참조로 사용한다.
- 세부 규칙(경로/파일명) 변경 시 단계 문서를 일괄 수정하지 않고 이 인덱스 우선 갱신한다.

## 핵심 규칙 문서

| 파일 | 목적 | 핵심 내용 |
|------|------|---------|
| CODE_ARCHITECTURE.md | 계층 구조 & 설계 원칙 | 모듈 계층, 의존성 방향, SOLID 원칙 |
| CODE_NAMING_RULE.md | 명명 규칙 | 클래스/함수/변수/패키지 명명 기준 |
| CODE_LIFE_CYCLE.md | 생명주기 관리 | Activity/Fragment/ViewModel 생명주기 규칙 |
| CODE_PATTERNS_INDEX.md | 패턴 전체 목록 | 9개 패턴 인덱스 → patterns/ |
