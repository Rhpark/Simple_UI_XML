# Step 2. 안전망 확인

리팩토링 전 현재 상태가 안전한가? (기준선 확보)

## 체크리스트

- [ ] 현재 빌드가 성공하는가?
- [ ] 기존 단위 테스트가 통과하는가?
- [ ] Robolectric 테스트가 통과하는가?
- [ ] 변경 전 현재 동작(입력/출력/부수효과)을 기록했는가?
- [ ] 테스트가 없는 대상이라면 최소한의 검증 방법을 준비했는가?

## 빌드/테스트 확인 명령

```bash
# 빌드 확인
./gradlew build

# 단위 테스트
./gradlew test

# Robolectric 테스트
./gradlew testRobolectric
```

## 심각도 기준

- CRITICAL: 빌드 실패 상태에서 리팩토링 진행
- HIGH: 테스트 미실행 상태에서 리팩토링 진행
- MEDIUM: 변경 전 동작 기록 없이 진행 (검증 불가)
