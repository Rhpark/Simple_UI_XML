# STEP_FORCE_STOP.md — 강제 중단 조건

이 파일은 analysis·review 두 워크플로우가 공유하는 강제 중단 규칙이다.

---

## 강제 중단과 복귀의 차이

| 구분 | 설명 | 행동 |
|---|---|---|
| **강제 중단** | 복귀해도 해소될 가능성이 없는 상황 | 작업 즉시 중단, 사용자에게 상태·원인·선택지 보고 |
| **복귀** | 특정 STEP을 재수행하면 해소 가능한 상황 | 해당 STEP으로 복귀 후 재수행, 이후 진행 |

---

## 강제 중단 적용 조건 (공통)

- 진입점 정보가 하나도 없다
- 핵심 파일이 0개다
- 루프 판정이 발생했다
- 요청이 워크플로우 범위 밖이고 사용자 질문 후에도 해소되지 않는다
