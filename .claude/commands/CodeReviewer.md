## 대상 코드 결정 우선순위
1. $ARGUMENTS 가 있으면 → 해당 파일 경로, 패키지명, 클래스명을 대상으로 한다.
2. $ARGUMENTS 가 없으면 → 현재 IDE에서 선택된 코드(ide_selection)를 대상으로 한다.

대상 코드를 다방면으로 분석 후 평가 해줘.
총 100점 으로 아래의 기준으로 평가 점수화 해줘.
- 개발 (*/20)
- 유지보수(*/20)
- 성능(*/20)
- 사용자 편의성(*/20)
- 기타(*/20)

잘한 부분, 못한 부분, 개선할 부분 등 알려줘.
코드가 복잡하니까 충분한 시간을 갖고 면밀히 분석하자.
