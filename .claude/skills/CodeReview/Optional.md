# Optional Steps

5단계 필수 리뷰 후 이상이 없다면, 사용자에게 아래 항목을 순서대로 문의한다.
각 항목은 사용자 응답에 따라 처리한다.

## 1. KDoc 주석

대상 코드에 KDoc 주석을 추가하시겠습니까?
- 예 → /KDoc 실행
- 아니오 → 다음 항목으로

## 2. 테스트 코드

대상 코드에 테스트 코드를 작성하시겠습니까?
- 예 → /TestCode 실행
- 아니오 → 다음 항목으로

## 3. 문서화 (PRD / SPEC / PLAN)

기능 문서를 작성/수정 하시겠습니까?
- 예 → (추후 예정 Skill)
- 아니오 → 리뷰 완료
